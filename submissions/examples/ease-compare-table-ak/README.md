# ⚡ Animated Feature Comparison Table Component

A production-ready product feature pricing matrix with fine-tuned entrance animations.

## ✨ What it does
Transforms flat informational tables into expressive user flows by combining staggered slide entry cues (`--row-index` translation offsets) with elastic boolean status vector pop triggers.

## 🚀 How to Use
```html
<div class="ease-compare-table-ak">
  <table>
    <thead>
      <tr>
        <th>Feature</th>
        <th class="ease-compare-table-ak__highlight">Pro</th>
      </tr>
    </thead>
    <tbody>
      <tr class="ease-compare-table-ak__row" style="--row-index: 0;">
        <td>Core Sync</td>
        <td><span class="ease-check-ak">✓</span></td>
      </tr>
    </tbody>
  </table>
</div>
