# ⚡ Animated Kanban Board Component

Productivity layout dashboard cell structure incorporating responsive grid cards.

## ✨ What it does
Provides a multi-column Kanban task board with smooth card lift transitions, priority indicators, active drop-zone status shading, and micro-scale counter indicators.

## 🚀 How to Use
```html
<div class="ease-kanban-ak">
  <div class="ease-kanban-ak__column">
    <div class="ease-kanban-ak__column-header">
      <h3>To Do</h3>
      <span class="ease-kanban-ak__count">2</span>
    </div>
  </div>
</div>
