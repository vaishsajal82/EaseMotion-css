# Ease File Attachment Preview

A smooth, interactive UI component for file attachments with fluid transition-based animations. Built strictly with HTML and pure CSS featuring custom cubic-bezier properties.

## Features
* **Staggered Entrance:** File elements gently slide up sequentially when mounted.
* **Micro-interactions:** File icons expand and zoom smoothly on container hover.
* **Ease Action Reveal:** Action controls (View, Download, Delete) slide out from an opacity state seamlessly using an elegant `cubic-bezier` dynamic curve.
* **Clutter-free UI:** Clean design layout hiding system utility buttons until user intent (hover) is shown.
