# Bridal E-commerce Landing Page

An elegant, romantic landing page for a Bridal Accessories Shop built with EaseMotion CSS.

## EaseMotion Classes Used
- `ease-fade-in`: Applied to Hero and Reviews sections for smooth entrance.
- `ease-slide-up`: Applied to product cards for an engaging reveal effect.
- `ease-hover-scale`: Applied to the CTA button for interactive feedback.

## Features
- Fully responsive grid layout.
- Romantic color palette (Champagne & Soft Pink).
- High-quality imagery for a professional look.

## How to Run
Simply open `demo.html` in any modern web browser. No compilation or server setup required.
