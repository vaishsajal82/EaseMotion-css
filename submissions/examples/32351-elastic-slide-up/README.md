# Elastic Slide Up Animation

## Overview

This submission demonstrates an **Elastic Slide Up** entrance animation built for EaseMotion CSS.

The animation smoothly moves an element upward while fading it into view. It includes a subtle elastic overshoot before settling into its final position, creating a polished and natural entrance effect.

## Features

- Smooth slide-up entrance animation
- Fade-in effect
- Elastic overshoot and settle motion
- Responsive demo layout
- Modern card-based UI
- Reusable CSS animation class

## Files

- `demo.html` – Demonstration page
- `style.css` – Animation and styling

## Use Cases

This animation is suitable for:

- Hero sections
- Landing pages
- Feature cards
- Call-to-action sections
- Dashboards
- Modal entrances

## Animation

The animation uses CSS keyframes with a custom cubic-bezier timing function to create a smooth elastic motion while keeping the interaction lightweight and visually appealing.
