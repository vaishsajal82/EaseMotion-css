# ⚡ Fix ease-glow Color Banding on Low Color Depth Displays

Provides seamless gradient dispersion for box-shadow animations on all monitor depths.

## ✨ What it does
Prevents harsh, stepped color ring artifacts (color banding) from forming on lower color profile environments (like 16-bit spaces, budget screens, or compression pipelines) during high-intensity neon glow operations.

## 🚀 How to Use
```html
<div class="ease-glow-smooth">
  Smooth glow without color banding
</div>
