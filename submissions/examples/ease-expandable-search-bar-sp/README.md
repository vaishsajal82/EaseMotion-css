# Expandable Search Bar

## What does this do?

This component creates an expandable search bar with a smooth animation. Clicking the search icon expands the input field, and pressing Enter submits the entered text.

## How is it used?

```html
<div class="search-container">
    <button class="search-btn">
        🔍
    </button>

    <input
        type="text"
        class="search-input"
        placeholder="Search..."
    >
</div>
```

## Why is it useful?

It provides a clean and modern search interface while saving screen space. The smooth animation improves user experience and fits well with EaseMotion CSS's animation-first philosophy.
