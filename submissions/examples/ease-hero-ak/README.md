# ⚡ Animated Hero Section with Floating Elements Component

A premium landing stage module incorporating hardware-accelerated parallax backplates.

## ✨ What it does
Creates an immersive, layered Hero presentation component featuring staggered text reveal transformations, a multi-stop color gradient flow loop, and decoupled, smooth background floating indicators.

## 🚀 How to Use
```html
<section class="ease-hero-ak">
  <div class="ease-hero-ak__floats">
    <div class="ease-hero-ak__float ease-hero-ak__float--1">⚡ Fast</div>
  </div>
  <div class="ease-hero-ak__content">
    <h1 class="ease-hero-ak__title">Animate Anything</h1>
  </div>
</section>
