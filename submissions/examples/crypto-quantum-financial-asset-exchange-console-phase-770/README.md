# Crypto-Quantum Financial Asset Exchange Console

## Overview

This folder contains a premium static showcase for a futuristic crypto-quantum financial asset exchange console. The experience is built as a direct-open HTML and CSS dashboard with high-contrast panels, inline SVG visuals, and CSS-only motion for a secure enterprise trading atmosphere.

## Features

- Fully static implementation using only `demo.html` and `style.css`
- No JavaScript, no build tools, and no external assets
- Inline SVG visuals for the quantum trading core, encrypted nodes, liquidity pools, transaction chains, market graphs, and digital security shield
- High-contrast enterprise color system aligned to the required palette
- Responsive dashboard layout for desktop, tablet, and mobile screens
- CSS-only animations for pulsing nodes, transaction flow, rotating encryption rings, graph motion, floating telemetry cards, hover lift, glow, and scanline sweep
