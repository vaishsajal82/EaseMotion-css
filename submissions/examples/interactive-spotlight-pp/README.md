# Interactive Mouse Spotlight Effect

## What does this do?

Adds a smooth interactive spotlight effect that follows the user's cursor, creating a subtle glowing highlight over an element.

---

## How is it used?

```html
<div class="spotlight-card">
    <h1>Build with EaseMotion</h1>
    <p>Move your mouse across this card.</p>
</div>
```

---

## Why is it useful?

This effect enhances user interaction by providing a modern, lightweight hover animation that can be used on hero sections, cards, feature panels, and other UI components. It aligns with EaseMotion CSS's animation-first philosophy while remaining reusable, visually appealing, and easy to integrate.
