# ⚡ Ease Animated Product Card with Quick View Component

Premium e-commerce interactive wrapper boasting modular action states.

## ✨ What it does
An elegant product presentation block optimized with interactive hover mechanics, clean image zoom transformations, and smooth, staggered action utilities.

```html
<div class="ease-product-card-ak">
  <div class="ease-product-card-ak__image-wrap">
    <img src="product.jpg" class="ease-product-card-ak__image" />
    <div class="ease-product-card-ak__overlay">
      <button class="ease-product-card-ak__quick-view">Quick View</button>
      <button class="ease-product-card-ak__wishlist">♡</button>
    </div>
  </div>
</div>
